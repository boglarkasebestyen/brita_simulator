# Brita Simulator 

### A Brita simulator written in SwiftUI ![](https://static.gosquared.com/images/software/logo_integ_swift_01.png)

This is my first SwiftUI app. After completing a beginner's-level Xcode / SwiftUI class, instead of finishing it off with a similar app that's been already covered in the course, I became inspired to write one that simulates a Brita, or any other water pitcher.

![](https://media.giphy.com/media/NaLazctnwt4XtUkkKl/giphy.gif)

### How it works

The simulator allows users to interact with it. 


* By default, the pitcher is filled with clean water, showing the water percentage at 100%.
* **"Pour water"** button: The "Pour water" button allows the user to decrease the water level by both percentage and on the graphic image.
* **"Fill tank"** button:  With the help of the "Fill tank" button the water tank gets filled with clean water showing an increase by both percentage and on the graphic image.


**Special features:** 

 1. At a random time interval, an alert warns the user to change the filter. When this happens, the water turns brown until the filter is changed with the help of the "Change filter" button. If the user chooses to change the filter, the water percentage is reset to zero and the user must fill the tank again with clean water. If the filter remains unchanged, filling the water tank or pouring water results in filling with / pouring dirty water. 
 2. Added bonus sound effects to accompany the buttons.
